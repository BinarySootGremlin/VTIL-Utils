# VTIL-Utils

VTIL command line utility.

## Building

```sh
mkdir build && cd build
cmake ..
```

## Usage

```
  vtil COMMAND {OPTIONS}

    VTIL command line utility

  OPTIONS:

      Commands
        dump                              Dump a .vtil file
        lift                              Lift a .vtil file
        opt                               Optimize a .vtil file
      Arguments
        -h, --help                        Display this help menu
```
